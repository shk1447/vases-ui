# Get Started

> vases UI에 대한 설치 및 사용 방법

## Install

```bash
# install locally (recommended)
npm install vases-ui --save
```

## CLI

```bash
# create vases-ui app
npx vases-ui create
```

## Usage

```javascript
import { Button } from 'vases-ui';

export const Page = () => {
  return <Button>Example</Button>;
};
```
